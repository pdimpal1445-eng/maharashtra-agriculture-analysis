# 🌾 Maharashtra Agriculture Analysis

## About
Analysis of 52 years Maharashtra crop 
and rainfall data using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Key Findings
1. Sugarcane = Maharashtra's top crop
   (153,430 tons average)
2. July rainfall weakly correlated 
   with rice production (0.31)
3. Onion area tripled since 1966
4. Rainfall varies 80mm to 392mm!
